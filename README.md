# 🪔 Diwali Sales Analysis Using Python

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data using Python.  
The main objective is to analyze customer purchasing behavior during the Diwali festival and identify key business insights.

The analysis helps understand:
- Customer demographics
- Sales trends
- High-performing states
- Product categories
- Occupation-based purchasing behavior
- Gender and age group insights

---

# 🎯 Business Problem
A retail company wants to understand customer purchasing patterns during Diwali sales to improve:
- Marketing strategies
- Customer targeting
- Product recommendations
- Sales performance

# 📈 Exploratory Data Analysis

The project includes analysis on:

## 👨‍👩‍👧 Gender Analysis
- Female customers purchased more products compared to males.
- Female customers generated higher revenue.

## 🎂 Age Group Analysis
- Customers aged 26–35 contributed the highest sales.

## 💍 Marital Status Analysis
- Married women were one of the highest purchasing groups.

## 🌍 State-wise Analysis
Top performing states:
- Uttar Pradesh
- Maharashtra
- Karnataka

## 💼 Occupation Analysis
Top occupations:
- IT Sector
- Healthcare
- Aviation

## 🛍️ Product Category Analysis
Top product categories:
- Clothing & Apparel
- Food
- Electronics

---

# 📌 Key Insights

- Female customers are the major buyers.
- Married women aged 26–35 years contributed most sales.
- Uttar Pradesh, Maharashtra, and Karnataka generated high revenue.
- IT, Healthcare, and Aviation professionals spent more during Diwali.
- Clothing and Food categories had high demand.

---

# ✅ Recommendations

- Target women customers with personalized marketing campaigns.
- Focus marketing on the 26–35 age group.
- Increase inventory for high-selling product categories.
- Run state-specific promotional campaigns.
- Provide special festive offers for working professionals.
